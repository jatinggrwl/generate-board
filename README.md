# ReactJS App

Projeto de um CRUD feito em ReactJS com API RESTful.

## Características

- CRUD
- API RESTful
- MUI-Datatables

## Requisitos
- NPM >= 6.4.1

## Tecnologias

- ReactJS

## Instalação

```
$ git clone https://github.com/danilomeneghel/reactjs_app_crud.git

$ cd reactjs_app_crud
$ npm install
$ npm start
```

## Demonstração

Para ver o App rodando, acesse:
